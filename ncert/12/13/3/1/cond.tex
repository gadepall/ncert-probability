The required probability is given by
\begin{align}
\pr{Y = 0} &= \pr{X = 0}\pr{Y = 0 \mid X = 0} + \pr{X = 1}\pr{Y = 0 \mid X = 1}\\
&= \brak{\frac{5}{10} \times \frac{7}{12}} + \brak{\frac{5}{10} \times \frac{5}{12}}
= \frac{1}{2}
\end{align}
