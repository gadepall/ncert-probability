    From the given information,
\begin{align}
    \pr{X = 0} = \frac{5}{9},
    \pr{X = 1} = \frac{4}{9}
    \\
    \pr{Y = 0 | X = 0} =  \frac{1}{2},
    \pr{Y = 0 | X = 1} =  \frac{5}{8}
\end{align}
The desired probability is
\begin{align}
\pr{Y = 0} &= \pr{X = 0}\pr{Y = 0 | X = 0} + \pr{X = 1}\pr{Y = 0 | X = 1}\\
           &= {\frac{5}{9}}\times{\frac{1}{2}} + {\frac{4}{9}}\times{{\frac{5}{8}}}
           = \frac{5}{9}
\end{align}
