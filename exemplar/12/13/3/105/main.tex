   Using De Morgan's law and axioms of probability,
\begin{align}
\pr{\brak{A+B}^\prime } &=\pr{A^\prime B^\prime }  \\
\pr{A^\prime  + B^\prime } &= \pr{A^\prime } + \pr{B^\prime } - \pr{A^\prime B^\prime }
\end{align}
Adding the above, 
\begin{align}
	\pr{A^\prime } + \pr{B^\prime }  = 1+\pr{A^\prime  + B^\prime }-\pr{A+B}
	=\frac{10}{9}
\end{align}
