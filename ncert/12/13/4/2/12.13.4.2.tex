\textbf{12.13.4.2 }An urn contains 5 red and 2 black balls. Two balls are randomly drawn. Let X represent the number of black balls. What are the possible values of X? Is X a random variable?\\
\textbf{Solution: }Possible values of X are as follows -
\begin{align}
	X = \{ 0, 1, 2 \}
\end{align}


A random variable is an assignment of real values to each outcome of the experiment. Therefore, X is an random variable.





Let $N = R + B$ and $n = r + b$\\
where,
\begin{table}[htbp]
	\centering
	\label{tab:Parameters}
	\input{tables/table.tex}
	\caption{Parameters Used}
\end{table}


then
\begin{align}
\pr{r, b} = \frac{ \comb{R}{r} \comb{B}{b} }{\comb{R + B}{r + b}}
\end{align}
\\
In our case ,\\
$
R = 5\\
B = 2\\
N = 5+2 = 7\\
$
and $n = 2$\\\\
Now,\\
$
n = r + b\\
\therefore 2 = r + b\\
\therefore r = 2 - b\\
$
\\
Now as $X = b$


and 
\begin{align}
\pr{r, b} = \frac{ \comb{R}{r} \comb{B}{b}}{\comb{R + B}{r + b}}
\end{align}
\begin{align}
\therefore \pr{X=b} = \frac{ \comb{5}{2-b} \comb{2}{b} }{\comb{7}{2}}
\end{align}



So,
Probability Distribution of X can be given as
\begin{align} 
p_X(b) = \frac{ \comb{5}{2-b} \comb{2}{b}}{21}
\end{align}
where b = \{0, 1, 2\}
